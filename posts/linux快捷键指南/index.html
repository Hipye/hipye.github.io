Linux快捷键指南

2023-09-06 • 更新于 2023-09-16 • 预计阅读时间 2 分钟

  1. 基本操作 Ctrl + C：中断当前正在运行的程序。
    Ctrl + D：结束当前会话或退出终端。
    Ctrl + Z：将当前程序置于后台运行。
    Ctrl + L：清屏。
    Tab：自动补全命令或文件名。

  2. 光标移动 Ctrl + A：将光标移至行首。
    Ctrl + E：将光标移至行尾。
    Ctrl + B：向后移动一个字符。
    Ctrl + F：向前移动一个字符。
    Alt + B：向后按单词移动。
    Alt + F：向前按单词移动。

  3. 命令历史： Ctrl + P：查看上一条命令。
    Ctrl + N：查看下一条命令。
    Ctrl + R：在历史命令中进行反向搜索。
    Ctrl + G：退出历史命令搜索模式。
    !!：执行上一条命令。
    !n：执行历史命令中的第n条命令。

  4. 文本编辑： Ctrl + U：删除光标之前的所有字符。
    Ctrl + K：删除光标之后的所有字符。
    Ctrl + W：删除光标之前的一个单词。
    Ctrl + Y：粘贴最近删除的文本。
    Ctrl + T：交换光标前两个字符。

  5. 窗口管理： Alt + Tab：在打开的窗口之间切换。
    Alt + F4：关闭当前窗口。
    Ctrl + Alt + L：锁定屏幕。

  6. 文件操作： Ctrl + N：新建文件或文件夹。
    Ctrl + O：打开文件。
    Ctrl + S：保存文件。
    Ctrl + X：剪切选中的文件或文本。
    Ctrl + C：复制选中的文件或文本。
    Ctrl + V：粘贴剪切或复制的文件或文本。
    Ctrl + Z：撤销上一步操作。
    Ctrl + Y：重做上一步撤销的操作。

  7. 终端操作： Ctrl + Shift + T：打开新终端选项卡。
    Ctrl + Shift + W：关闭当前终端选项卡。
    Ctrl + Alt + F1~F6：切换至虚拟终端。
    Ctrl + Alt + F7：返回图形界面虚拟终端。

  8. 进程管理： Ctrl + Z：将前台运行的程序放入后台。
    jobs：查看当前所有作业。
    fg %n：将作业 n 切换至前台运行。
    bg %n：将作业 n 切换至后台运行。
